# EasyDB
![EasyDBLogo](https://github.com/Tamuel/EasyDB/blob/master/EasyDB/src/Image/easyDB.png)\
Create DB query easier and simpler.\
DB쿼리를 쉽게 만들고 눈으로 볼 수 있도록 제작한 초심자용 프로그램입니다.\
